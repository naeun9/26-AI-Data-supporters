## 창업메이트 유형 검사 분류 기준

### 1. 분류 방식 개요

본 유형 검사는 12개의 문항에 대한 사용자의 응답을 기반으로, 가장 높은 빈도로 선택된 성향을 최종 결과로 도출하는 '최다 빈도수(다수결) 측정 방식'을 채택하고 있습니다.

### 2. 매핑 (Mapping) 구조

문항 및 선택지: 총 12개의 질문으로 구성되며, 각 질문당 4개의 선택지가 제공됩니다.

유형 연결: 각 선택지는 5가지 창업 유형(기술혁신형, 아이디어형, 생계형자영업형, 경력전환형, 재도전형) 중 1개의 유형과 1:1로 매핑되어 있습니다.
(참고 파일: frontend/src/data/quiz.ts)

### 3. 점수 산정 및 결과 도출 프로세스

점수 누적: 사용자가 특정 선택지를 클릭할 때마다 해당 선택지와 연결된 유형의 점수(Count)가 1점씩 가산됩니다.

합산: 12번 문항까지 모든 응답이 완료되면, 5개 유형에 누적된 최종 점수를 합산하여 비교합니다.

결과 판정: 가장 높은 점수를 획득한 유형을 사용자의 최종 창업 유형으로 판정합니다.

### 4. 예외 처리 (동점자 발생 시)

최고 점수를 획득한 유형이 2개 이상(동점)일 경우, 코드 내에 사전 정의된 배열 우선순위(Index Order)에 따라 최종 결과를 결정합니다.

우선순위 서열:

1. 기술혁신형 (tech)

2. 아이디어형 (idea)

3. 생계형자영업형 (mainstreet)

4. 경력전환형 (career)

5. 재도전형 (comeback)

(예시: 아이디어형과 기술혁신형이 동일한 최고 점수를 기록할 경우, 우선순위가 높은 '기술혁신형'이 최종 결과로 도출됨)
